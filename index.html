Under the Hood: C# & Electron Bridge

TypeBar uses a hybrid micro-architecture to deliver speed and accessibility at the same time.

01. Key Hook

Native Win32 C#

A lightweight C# console process registers a low-level Win32 keyboard hook (`SetWindowsHookEx`) to capture and intercept physical keystrokes system-wide. This ensures 0.5ms response time, bypassing standard OS bottlenecks.

02. IPC Streams

IPC Bridge

Inter-process communication pipes stream keyboard event data directly into the Electron main thread via standard streams. When hotkeys are intercepted (like `Tab` or `Right Arrow`), they are suppressed in C# so standard apps ignore them while TypeBar selects suggestions.

03. UI Render

React Overlay

An Electron viewport renders the react overlay as a borderless, transparent click-through window positioned above the active desktop workspace. High-performance styling handles frosted glass blur and cyberpunk glowing borders smoothly.

Features Designed for Speed

Built from the ground up for high-performance typists and gamers.

Typo Auto-Correction

Instantly corrects accidental slips like 'teh' to 'the' or 'dont' to 'don't'. Runs on-the-fly and seamlessly replaces characters in your target apps.

Smart Word Completion

Predicts words dynamically before you finish typing. Powered by a tech-aware lexicography dictionary matching prefix patterns.

Adaptive Emoji Lookup

No more hunting through system emoji grids. Simply start typing words like 'happy', 'fire', 'dog', or 'money' and immediately tab in matching emojis.

Mechanical Click Synthesis

Enjoy acoustic feedback using synthesizers modulated inside Web Audio APIs. Simulates crisp mechanical switch clicks with every stroke.

Active App Detection

Identifies active windows automatically (VSCode, Discord, Chrome, Roblox). Resets buffer logs dynamically when jumping processes to ensure context clarity.

Tray Configurator

Adjust window transparency, blur, custom hotkeys, layout positions, and speed rates easily via the expanded native settings drawer.

Keyboard Hotkeys & Controls

Take full advantage of TypeBar layout controls without taking your hands off the keyboard.

Action Shortcut Binding Description
Toggle Visibility Ctrl + Alt + S Briefly show or hide the overlay layout on your screen.
Accept First Suggestion Tab or F1 Inserts the auto-correct or first prediction capsule.
Accept Second Suggestion Right Arrow or F2 Inserts the second prediction capsule.
Select Capsule (3 - 5) F3 · F4 · F5 Selects the third, fourth, or fifth capsule respectively.
Quick Phrase Templates F1 to F5 (When buffer empty) Inserts quick templates like 'wsg', 'help', 'ggs'.
Emergency Deactivation Ctrl + Alt + Shift Immediately kills keyboard hooks if app hangs.
